安装方法

三步完成安装，即刻开始使用

01

安装 Fabric

确保已安装 Minecraft 1.20.1 及 Fabric Loader。前往 fabricmc.net 下载对应版本的安装器。

> Fabric Loader 1.20.1 ✔
02

下载模组

下载 SysCmd v1.3 模组文件（约 0.1MB），将其放入 .minecraft/mods 文件夹中。

> mc2cmd-1.3.jar → mods/
03

启动游戏

启动 Minecraft，进入世界后使用 /cmd 或 /ps 命令即可开始使用。

MC> /cmd dir ✔

注意事项

使用前请务必了解以下内容

⚠ 注意事项
  • 本模组仅在 Windows 系统上可用，不支持 macOS / Linux
  • 执行的命令拥有当前用户的系统权限，请谨慎操作
  • 部分命令可能需要管理员权限才能正常执行
  • 长时间运行的命令可能导致游戏短暂卡顿
  • 建议在单人模式或自己的服务器中使用
⛔ 禁止用途
  • 严禁用于破坏他人计算机系统或窃取数据
